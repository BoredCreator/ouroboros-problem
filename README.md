# The Ouroboros Problem

Minimal placeholder site for a project on AI, fair use, and the loop that feeds itself — human work feeds the model, the model feeds its imitators, and the copies flood the internet the original learned from.

A single self-contained `index.html`: an animated ouroboros (a slowly rotating ring of tapering dots with a peristaltic wave, mouth open around a red tail tip) in the project's black / dark-red / white palette. No dependencies, no build step. Respects `prefers-reduced-motion`.
