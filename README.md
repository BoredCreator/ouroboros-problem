# The Ouroboros Problem

A project site on AI, fair use, and the loop that feeds itself — human work feeds the model, the model feeds its imitators, and the copies flood the internet the original learned from.

Eleven pages behind a numbered navigation toolbar (Home · Dilemma · Thesis · Frameworks · four Supporting Arguments · Rebuttal · Conclusion · Works Cited). The home page carries the animated ouroboros — a slowly rotating ring of tapering dots with a peristaltic wave, mouth open around a red tail tip — and Page 2 holds the structural-dilemma and why-it-matters proposal paragraphs with the embedded "It's Fair Use When We Do It." infographic. Plain HTML + one stylesheet, no dependencies, no build step. Respects `prefers-reduced-motion`.
